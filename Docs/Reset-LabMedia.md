# Reset-LabMedia
## SYNOPSIS
Reset the lab media entries to default settings.

## SYNTAX

```
Reset-LabMedia [-WhatIf] [-Confirm]
```

## DESCRIPTION
The Reset-LabMedia removes all custom media entries, reverting them to default values.
