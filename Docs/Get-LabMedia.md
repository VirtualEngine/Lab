# Get-LabMedia
## SYNOPSIS
Gets registered lab media.

## SYNTAX

```
Get-LabMedia [[-Id] <String>] [-CustomOnly]
```

## DESCRIPTION
The Get-LabMedia cmdlet retrieves all built-in and registered custom media.
