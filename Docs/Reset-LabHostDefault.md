# Reset-LabHostDefault
## SYNOPSIS
Resets lab host default settings to default.

## SYNTAX

```
Reset-LabHostDefault [-WhatIf] [-Confirm]
```

## DESCRIPTION
The Reset-LabHostDefault cmdlet resets the lab host's settings to default values.
