# Unregister-LabMedia
## SYNOPSIS
Unregisters a custom media entry.

## SYNTAX

```
Unregister-LabMedia [-Id] <String> [-WhatIf] [-Confirm]
```

## DESCRIPTION
The Unregister-LabMedia cmdlet allows removing custom media entries from the host's configuration.
